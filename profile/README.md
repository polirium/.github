## Về Polirium

Polirium là một nền tảng **ERP (Enterprise Resource Planning)** theo kiến trúc module hóa, giúp doanh nghiệp xây dựng và tích hợp nhanh các nghiệp vụ riêng.

- Core system tập trung vào khả năng mở rộng.
- Dễ tùy biến theo quy mô và quy trình vận hành.
- Tối ưu cho tốc độ phát triển tính năng theo từng module.

## Repository nổi bật

- [`polirium/polirium`](https://github.com/polirium/polirium): Nền tảng ERP chính.
- [`polirium/platform-core`](https://github.com/polirium/platform-core): Core của hệ sinh thái Polirium.
- [`polirium/polirium.com`](https://github.com/polirium/polirium.com): Website/landing.
- [`polirium/polirium-tabler-datatable`](https://github.com/polirium/polirium-tabler-datatable): Thành phần datatable.
- [`polirium/laravel-impersonate`](https://github.com/polirium/laravel-impersonate): Gói impersonate fork.
